# Settler Business Strategy (Internal)

**Confidential - Internal Use Only**  
**Version:** 1.0  
**Last Updated:** 2024-01-01

## Executive Summary

Settler operates a **dual-product, dual-license business model**:
1. **Settler API** - Proprietary SaaS for reconciliation infrastructure
2. **React.Settler** - OSS protocol + Commercial features for UI development

**Goal:** Profitable by Month 12, $1M+ ARR by Year 2, $4M+ ARR by Year 3

## Business Model

### Product Portfolio

**Settler API (Proprietary SaaS)**
- **License:** Proprietary (Closed Source)
- **Pricing:** Usage-based + subscription ($99-$499/month)
- **Target:** Production businesses
- **Revenue:** $50K ARR (Year 1) → $2M ARR (Year 3)

**React.Settler (OSS + Commercial)**
- **OSS License:** MIT (Free Forever)
- **Commercial License:** $99/month or $990/year
- **Enterprise License:** Custom ($5K-$50K+/year)
- **Target:** Developers building reconciliation UIs
- **Revenue:** $120K ARR (Year 1) → $1.2M ARR (Year 3)

### Revenue Mix (Year 1)

- Settler API SaaS: 30% ($50K ARR)
- React.Settler Commercial: 70% ($120K ARR)
- **Total:** $170K ARR

### Revenue Mix (Year 3)

- Settler API SaaS: 48% ($2M ARR)
- React.Settler Commercial: 29% ($1.2M ARR)
- React.Settler Enterprise: 24% ($1M ARR)
- **Total:** $4.2M ARR

## Customer Acquisition

### Acquisition Channels

1. **Product-Led Growth (60% of customers)**
   - OSS adoption → Commercial upgrade
   - Free trial → Paid subscription
   - In-app upgrade prompts

2. **Content Marketing (20% of customers)**
   - Blog posts
   - Tutorial videos
   - Case studies
   - SEO optimization

3. **Community & Partnerships (10% of customers)**
   - GitHub community
   - Discord server
   - Partner integrations
   - Referral program

4. **Sales (10% of customers, Enterprise)**
   - Outbound prospecting
   - Inbound leads
   - Partner referrals
   - Conference networking

### Customer Acquisition Cost (CAC)

- **OSS → Commercial:** $100-$300 (low-touch, self-serve)
- **Trial → Paid (Settler API):** $300-$500 (self-serve)
- **Enterprise:** $5,000-$15,000 (high-touch, sales-assisted)

### Lifetime Value (LTV)

- **Commercial:** $3,000-$15,000 (2-5 year average lifetime)
- **Settler API SaaS:** $10,000-$50,000 (3-5 year average lifetime)
- **Enterprise:** $50,000-$250,000 (5+ year average lifetime)

### LTV:CAC Ratios

- **Commercial:** 10:1 to 50:1
- **Settler API SaaS:** 20:1 to 100:1
- **Enterprise:** 3:1 to 10:1

## Pricing Strategy

### Price Positioning

- **OSS:** Free (loss leader, drives adoption)
- **Commercial:** $99/month (value anchor, under $100)
- **Settler API Starter:** $99/month (competitive)
- **Settler API Pro:** $499/month (premium)
- **Enterprise:** Custom (value-based)

### Value-Based Pricing Rationale

**Commercial ($99/month):**
- Saves 2-4 weeks of development time
- Value: $5,000-$10,000 (developer time)
- ROI: 50x-100x

**Settler API Pro ($499/month):**
- Saves 1-2 FTE (Full-Time Equivalent) in finance team
- Value: $50,000-$100,000/year (salary + benefits)
- ROI: 100x-200x

**Enterprise (Custom):**
- Saves $100K+/year in manual reconciliation
- Value: $100,000-$500,000/year
- ROI: 10x-50x
