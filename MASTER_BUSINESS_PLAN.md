# Settler Master Business Plan

**Version:** 1.0  
**Last Updated:** 2024-01-01  
**Status:** Active

## Executive Summary

Settler operates a **dual-product, dual-license business model** designed for sustainable growth and profitability:

1. **Settler API** - Proprietary SaaS for reconciliation infrastructure
2. **React.Settler** - OSS protocol + Commercial features for UI development

**Mission:** Make reconciliation effortless for developers and businesses worldwide.

**Vision:** Become the industry standard for reconciliation workflows.

**Target:** Profitable by Month 12, $1M+ ARR by Year 2, $4M+ ARR by Year 3.

## Business Model

### Product Portfolio

#### Settler API (Proprietary SaaS)
- **License:** Proprietary (Closed Source)
- **Pricing:** Usage-based + subscription ($99-$499/month)
- **Target:** Production businesses needing reconciliation infrastructure
- **Value Prop:** Managed service, infrastructure, support
- **Revenue:** $50K ARR (Year 1) → $2M ARR (Year 3)

#### React.Settler (OSS + Commercial)
- **OSS License:** MIT (Free Forever)
- **Commercial License:** $99/month or $990/year
- **Enterprise License:** Custom ($5K-$50K+/year)
- **Target:** Developers building reconciliation UIs
- **Value Prop:** Declarative React components, backend-agnostic protocol
- **Revenue:** $120K ARR (Year 1) → $2.2M ARR (Year 3)

### Revenue Mix

**Year 1:**
- Settler API SaaS: 30% ($50K ARR)
- React.Settler Commercial: 70% ($120K ARR)
- **Total:** $170K ARR

**Year 3:**
- Settler API SaaS: 48% ($2M ARR)
- React.Settler Commercial: 29% ($1.2M ARR)
- React.Settler Enterprise: 24% ($1M ARR)
- **Total:** $4.2M ARR

## Customer Acquisition

### Funnel Stages

1. **Awareness** - GitHub, npm, content marketing
2. **Consideration** - Documentation, playground, community
3. **Evaluation** - Free trial, OSS usage, demos
4. **Purchase** - Self-serve signup, sales team
5. **Expansion** - Usage upsells, feature upgrades

### Acquisition Channels

- **Product-Led Growth:** 60% (OSS → Commercial, Trial → Paid)
- **Content Marketing:** 20% (Blog posts, tutorials, SEO)
- **Community & Partnerships:** 10% (GitHub, Discord, referrals)
- **Sales:** 10% (Enterprise, high-touch)

### Conversion Rates

- OSS → Commercial: 5-10%
- Trial → Paid: 10-15%
- Demo → Enterprise: 20-30%

### Customer Acquisition Cost (CAC)

- OSS → Commercial: $100-$300
- Trial → Paid: $300-$500
- Enterprise: $5,000-$15,000

### Lifetime Value (LTV)

- Commercial: $3,000-$15,000
- Settler API SaaS: $10,000-$50,000
- Enterprise: $50,000-$250,000

### LTV:CAC Ratios

- Commercial: 10:1 to 50:1
- Settler API SaaS: 20:1 to 100:1
- Enterprise: 3:1 to 10:1
