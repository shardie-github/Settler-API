## Conversion Paths

### Path 1: OSS → Commercial (Self-Serve)

1. **Discover OSS** on GitHub/npm
2. **Use free features** to build prototype
3. **Hit commercial feature** (e.g., Shopify integration)
4. **See upgrade prompt** in UI
5. **Click "View Pricing"** → Pricing page
6. **Sign up** → $99/month subscription
7. **Get license key** → Set in app
8. **Unlock features** → Full Commercial access

**Timeline:** 1-4 weeks  
**Conversion Rate:** 5-10%  
**CAC:** $100-$300

### Path 2: Trial → Paid (Settler API)

1. **Discover Settler API** via content/search
2. **Sign up for free trial** (14 days)
3. **Connect systems** and run reconciliation
4. **See value** (time saved, accuracy improved)
5. **Trial ending email** → Upgrade prompt
6. **Choose tier** → Starter ($99/month) or Pro ($499/month)
7. **Enter payment** → Subscription activated
8. **Onboarding** → Best practices, support

**Timeline:** 2-4 weeks  
**Conversion Rate:** 10-15%  
**CAC:** $300-$500

### Path 3: Enterprise (Sales-Assisted)

1. **Enterprise need** identified (large org, compliance requirements)
2. **Request demo** → Sales team contact
3. **Discovery call** → Understand requirements
4. **Custom demo** → Showcase enterprise features
5. **POC** → 30-day proof of concept
6. **Proposal** → Custom pricing and terms
7. **Contract negotiation** → Legal review
8. **Onboarding** → Dedicated support, implementation

**Timeline:** 2-6 months  
**Conversion Rate:** 20-30%  
**CAC:** $5,000-$15,000

## Key Conversion Points

### 1. Upgrade Prompts (OSS → Commercial)

**Triggers:**
- User tries Commercial feature
- User reaches usage limit
- User shows intent (multiple feature attempts)

**Messaging:**
- Clear value proposition
- Feature comparison
- Pricing transparency
- Easy upgrade path

**Placement:**
- In-app prompts
- Feature gates
- Documentation links
- Email campaigns

### 2. Trial Ending (Trial → Paid)

**Triggers:**
- 3 days before trial ends
- 1 day before trial ends
- Trial expired

**Messaging:**
- Value recap (what you accomplished)
- ROI calculation (time saved)
- Limited-time offer (if applicable)
- Support availability

**Placement:**
- In-app banners
- Email sequences
- Dashboard notifications

### 3. Usage Limits (Starter → Pro)

**Triggers:**
- 80% of usage limit reached
- 100% of usage limit reached
- Feature unlock needed

**Messaging:**
- Usage dashboard
- Upgrade benefits
- Cost per additional unit
- Automatic upgrade option

## Pricing Strategy

### Price Anchoring

- **OSS:** Free (anchor)
- **Commercial:** $99/month (value anchor)
- **Enterprise:** Custom (premium anchor)

### Value-Based Pricing

- **Commercial:** Price based on value (saves 2-4 weeks dev time = $5K-$10K)
- **Enterprise:** Price based on ROI (saves $50K+/year in manual work)

### Psychological Pricing

- **$99/month:** Under $100 threshold
- **$990/year:** 17% discount (annual)
- **Custom:** Negotiate based on value
